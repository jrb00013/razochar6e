# Shell completions

Regenerate after CLI changes:

```bash
make completions
```

Install examples:

```bash
# bash
sudo cp completions/razochar6e.bash /etc/bash_completion.d/razochar6e

# zsh
sudo cp completions/_razochar6e /usr/local/share/zsh/site-functions/

# fish
cp completions/razochar6e.fish ~/.config/fish/completions/
```
