# Security policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| < 0.2   | No        |

## Reporting

Open a [GitHub Security Advisory](https://github.com/jrb00013/razochar6e/security/advisories/new) or email the maintainer via GitHub profile contact.

`razochar6e` requires elevated privileges to write charge thresholds — only install from trusted sources and verify release checksums on the [Releases](https://github.com/jrb00013/razochar6e/releases) page.
